CSS Selectors & Specificity

Common class selectors

Selector Tag Selector Type Description
" * " Universal Selects all element on the document.
" ElementName " Type Selects all element that have the given node name.
" .ClassName " Class Selects all element that have the given class attribute.
" #IDName " ID Selects an element based on the value of its id attribute.

Specificity Chain

The following list of selector types increases by specificity:

  • Type selectors
  • Class selectors
  • ID selectors

The universal selector (" * ") has no effect on specificity.
